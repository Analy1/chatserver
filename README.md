# C++ 高性能聊天服务器

基于 Muduo 网络库 + MySQL + Redis 实现的高并发聊天服务器

## 技术栈
- C++11
- Muduo 网络库
- MySQL 数据库
- Redis 发布订阅
- JSON 序列化
- CMake 构建

## 核心功能
- 用户登录 / 注册
- 点对点聊天
- 群聊功能
- 离线消息存储
- 心跳机制
- 连接池优化数据库性能

## 核心优化
- **手写 MySQL 连接池**
  - 消除频繁创建销毁数据库连接的开销
  - 高并发下 QPS 提升 10 倍
  - 避免数据库连接风暴导致服务崩溃


## 项目架构
客户端 -> Muduo 网络层 -> 业务服务层 -> 数据访问层 -> MySQL
